# Assets - Models

Esta pasta deve conter:
- `jordan4.glb` - Modelo 3D do Air Jordan 4 Pure Money para visualização nos drops especiais

Para adicionar o modelo 3D:
1. Coloque o ficheiro `Air Jordan 4 Pure Money.glb` nesta pasta
2. Renomeie para `jordan4.glb`
3. Os produtos com `tipo: "drop"` e `modelUrl: "assets/models/jordan4.glb"` irão mostrar o modelo 3D interativo
